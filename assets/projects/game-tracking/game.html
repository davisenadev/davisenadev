About the Project

What it is

Game Tracking is a web page built to help gamers keep track of their gaming progress across multiple platforms. Whether you play on PS4, Xbox, PC or Nintendo, you can register your games, mark your status and maintain an organized library.

The project focuses on delivering a clean, intuitive interface that makes it easy to manage and visualize your gaming collection without friction.

Technologies

  • Angular 17
  • TypeScript
  • NestJS
  • Node.js
  • PostgreSQL

Features

Multi-platform

Track games across PS4, Xbox, PC and Nintendo from a single dashboard.

Progress Tracking

Mark games as playing, completed, on hold or wishlist.

Organized Library

Browse and search your whole collection in an organized visual layout.

Clean Design

Intuitive and responsive interface built for a great user experience.
