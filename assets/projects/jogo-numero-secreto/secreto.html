About the Project

What it is

Secret Number Game is a simple web game where the player must guess a randomly generated number.

After each guess, the game informs whether the secret number is higher or lower, helping the player reach the correct answer.

Technologies

  • HTML
  • CSS
  • JavaScript

Features

Random Number

Generates a secret number for the player to guess.

Hints

Indicates if the guess is higher or lower than the correct number.

Restart Game

Allows the player to restart and play again.
